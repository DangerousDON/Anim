  <h1 align="center">Anim</h1>

Anim allows you to use animations very easily. You can use it in your UIKit application for make smooth animations, using Swift. 


<h1 align="center">Features</h1>

 - Position (CGPoint)
 - Bounce effect
 - Resize (CGSize)
 - Rotation
 - Rotation X
 - Rotation Y
 - Rotation Z
 - Fade
 - Border raduis
 - Move circle
 - Animations sequence
 - Repeat animations
